# Sacred Smoke

A modern, visually stunning landing page featuring animated backgrounds, smooth transitions, and a clean responsive design.

## Features

- **Animated Particle Background**: Floating particle effects that create an immersive atmosphere
- **Glowing Orb Effects**: Dynamic pulsing orbs that add depth to the visual experience
- **Responsive Navigation**: Clean, fixed navigation bar with smooth hover effects
- **Modern Hero Section**: Eye-catching gradient text and call-to-action buttons
- **Feature Cards**: Elegant cards with hover animations showcasing key features
- **Mobile Responsive**: Fully responsive design that works on all device sizes

## Getting Started

Simply open `index.html` in a web browser to view the landing page.
